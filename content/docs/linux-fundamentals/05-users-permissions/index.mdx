# Section 05 — Users & Permissions

<Callout type="info">
  Linux একটা multi-user system। প্রতিটা user-এর আলাদা identity, আলাদা permissions। এই section-এ শিখবে users কীভাবে manage হয়, sudo কীভাবে কাজ করে, আর SSH দিয়ে remote server-এ কীভাবে connect করে।
</Callout>

এই section শেষ করলে তুমি জানবে:

- Linux-এ users আর groups কীভাবে কাজ করে
- `sudo` আর `su` দিয়ে elevated access কীভাবে নেওয়া যায়
- SSH দিয়ে remote server-এ securely connect করতে
