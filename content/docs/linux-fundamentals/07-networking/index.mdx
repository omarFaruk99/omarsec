# Section 07 — Networking

<Callout type="info">
  Cybersecurity-র বেশিরভাগ কাজ network নিয়ে। Target machine-এর IP বের করা, কোন ports open আছে, কোন services চলছে, shell কীভাবে পাবে — সব Linux terminal থেকে হয়। এই section-এ সেই tools শিখবে।
</Callout>

এই section শেষ করলে তুমি জানবে:

- Machine-এর IP address, interfaces, connectivity check করতে
- Open ports আর running services দেখতে
- Terminal থেকে HTTP requests পাঠাতে
- Netcat দিয়ে port check, file transfer, এবং reverse shell নিতে

- [Netcat](/docs/linux-fundamentals/07-networking/netcat) — port check, banner grab, file transfer, reverse shell
