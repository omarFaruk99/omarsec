# Section 08 — Package Management

<Callout type="info">
  Linux-এ software install করা Windows-এর মতো installer file খোঁজার বিষয় না। Package manager দিয়ে একটাই command-এ install, update, remove হয়। এই section-এ Ubuntu/Debian-এর `apt` এবং `.deb` file-এর জন্য `dpkg` শিখবে।
</Callout>

এই section শেষ করলে তুমি জানবে:

- Software install, update, remove করতে
- `apt update` আর `apt upgrade` এর পার্থক্য (common confusion)
- Package খুঁজতে
- `.deb` file manually install করতে

- [dpkg](/docs/linux-fundamentals/08-package-management/dpkg) — `.deb` file manually install করা
