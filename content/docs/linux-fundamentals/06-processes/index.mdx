# Section 06 — Processes

<Callout type="info">
  তুমি যতটা command বা program চালাও, সেটাই একটা process। Linux-এ সব কিছু process — background services, network listeners, তোমার terminal। এই সেকশনে আমরা প্রসেসের বেসিক থেকে শুরু করে সিস্টেম ইন্টারনালস এবং অটোমেশন শিখবো।
</Callout>

এই section শেষ করলে তুমি জানবে:

- প্রসেসের ইন্টারনাল আইডেন্টিটি (UID, EUID) এবং `/proc` ফাইলসিস্টেম
- Background-এ jobs চালাতে আর manage করতে
- Unwanted processes বন্ধ করতে এবং সিগন্যাল কন্ট্রোল করতে
- কাস্টম Systemd সার্ভিস তৈরি এবং ম্যানেজ করতে
- Cron এবং Systemd Timers ব্যবহার করে টাস্ক অটোমেট করতে
- Nice এবং Renice ব্যবহার করে প্রসেসর প্রায়োরিটি কন্ট্রোল করতে
