# Section 06 — Processes

<Callout type="info">
  তুমি যতটা command বা program চালাও, সেটাই একটা process। Linux-এ সব কিছু process — background services, network listeners, তোমার terminal। এগুলো দেখতে আর control করতে পারাটা সিস্টেম বোঝার জন্য এবং cybersecurity-র জন্য অপরিহার্য।
</Callout>

এই section শেষ করলে তুমি জানবে:

- System-এ কোন processes চলছে সেটা দেখতে
- Background-এ jobs চালাতে আর manage করতে
- Unwanted processes বন্ধ করতে
