# Linux

Linux is the backbone of cybersecurity. Almost every server, firewall, and security tool runs on Linux. Mastering it is non-negotiable for any pentester.

## Why Linux for Security?

- Most servers in the world run Linux
- All major security tools (Nmap, Burp Suite, Metasploit) are built for Linux
- Kali Linux is the industry-standard pentesting OS
- Understanding Linux internals helps you find vulnerabilities

## What You'll Learn

In this section, we cover the essentials:

- **Basic Commands** — Navigate the terminal like a pro
- **File System** — Understand where everything lives
- **Users & Permissions** — How Linux controls access
- **Process Management** — Monitor and control running processes

## Prerequisites

- A Linux machine (Ubuntu, Kali, or any distro)
- Access to a terminal
- Curiosity and patience

> Start with [Basic Commands](/docs/linux/basic-commands) if you're completely new.
