# OmarSec — Cybersecurity Knowledge Base

Welcome to the official repository for **[omarsec.com](https://www.omarsec.com)**.

This project is a personal cybersecurity knowledge base where I document my journey across Pentesting, Linux administration, and Server engineering. It is designed with a minimalist, "SWR-inspired" aesthetic using modern web technologies.

## 🚀 Live Site
Check out the live documentation and labs at: **[https://omarsec.com](https://www.omarsec.com)**

## 🛠️ Tech Stack
- **Framework**: Next.js 16 (Turbopack)
- **Engine**: Nextra v4 (Markdown/MDX based)
- **Styling**: Vanilla CSS / Geist Mono & Geist Typography
- **Deployment**: Vercel (CI/CD via GitHub)
- **Performance**: High SEO optimization and dark-mode by default

## 📂 Project Structure
- `content/index.mdx` — The minimal landing page.
- `content/docs/` — All documentation modules (Git & GitHub, etc.).
- `app/layout.jsx` — Core layout shell with custom SWR-style footer.
- `app/globals.css` — Modern branding and typography rules.

## 💻 Running Locally

To run the project in your local development environment:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/omarFaruk99/omarsec.git
   cd omarsec
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```
   Open `http://localhost:3000` to see the result.
